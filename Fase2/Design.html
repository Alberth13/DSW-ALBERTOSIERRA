Propósito del Sitio

Este sitio web, tiene como proposito brindar al lector una breve explicación de los temas: Flexbox, Listas y Enlaces. Dentro del tema de diseño de sitios web.

Flexbox: Es una herramienta de creación de diseño CSS, con la cual se puede tener un mayor control de los espacios, justificación y alineación, dentro de un contenedor. Teniendo como base el uso de ejes para su desplazamiento ya sea horizontal (por defecto) o vertical, depediendo de la propiedad flex-direction. Estos ejes son paralelos el uno del otro (eje principal y eje trasversal o cruzado) y sobre el eje principal van los flex items. Eje principal: es definido por flex-direction y tiene cuatro valores, row, row-reverse, column y column- reverse. Asi si trabajamos row el desplazamiento sera por la fila y si trabajamos column el desplazamiento se hará por las columnas.

Listas: En html se emplean tres tipos de etiquetas para generar listas. Estan pueden ser ordenadas, desordenadas y de definiciones. Las ordenadas van con la etiqueta ol /ol y dentro de esta li /li que sirve para indicarnos las lineas de texto a organizar, tal como lo haria una viñeta o numeración del mismo. Las desordenadas van con la etiqueta ul /ul y li /li de la misma manera que el anterior tipo. las de definiciones son mas estilizadas, van con la etiqueta dl /dl, luego dt /dt para el término a definir y dd /dd para la definición del término.

Enlaces: Sirven para enlazar una pagina web, una imagen, un texto, etc a uno o más sitios web. se hace con la etiqueta "a" la cual tiene como atributos principales href y name. href indica la dirección de apuntamiento del enlace URL y name es para ponerle un nombe al enlace.
